# 🧠 Face Recognition -

## 📌 Overview
A machine learning project for face detection and recognition using image classification.

## 📁 Project Structure
- `data/` - Training datasets (face & non-face images)
- `notebooks/` - Jupyter notebooks with model code

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Machine Learning / Computer Vision

## 📊 Dataset
- Face images: `archive (7).zip`
- Non-face images: `non_face.zip`
